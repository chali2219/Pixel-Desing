# Pixel Design Portfolio

This is a complete, multi-page portfolio website for a digital artist.

## How to Run
Simply double-click on `index.html` to open the website in your browser. No installation or Node.js required!

## Features
- **Modern Design**: Dark theme with neon accents (Cyberpunk/Digital Art style).
- **Glassmorphism**: Translucent cards and navigation.
- **Responsive**: Works on mobile and desktop.
- **Gallery**: Filterable portfolio section.
- **Contact Form**: Ready-to-use design.

## Customization
- **Images**: The site currently uses placeholder images from the internet. To use your own art:
  1. Save your images in the `assets` folder.
  2. Open the HTML files in a text editor (like Notepad or VS Code).
  3. Replace the `src="https://..."` links with `src="assets/your-image.jpg"`.

## Technologies
- HTML5
- Tailwind CSS (via CDN)
- JavaScript (for interactions)

Enjoy your new website!
